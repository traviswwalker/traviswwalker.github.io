Graphs

All graphs that are put in any publication, report, or presentation, including any data that you send to me to see, must be made in MATLAB until further notice. Assuming that the data is obtained from an external measurement, the figure must be accompanied by a data file (e.g., .csv, .xlsx, etc.) and the associate m-file when providing me the figure. This m-file needs to produce the exact figure that you are sending without any GUI interaction.

Papers

All articles will be named as exemplified by walker2015multiphase.pdf, where we will use the last name of the first author, the year of publication, and the first significant word of the title to designate the file. This designation will also be used for our citations.

Publications

All publications will be completed in LaTeX. Unless planning a submission to a specific journal, we will default to use the revtex document class that is used by AIP, APS, and others.

If you are using a Mac, TexShop is a relatively easy way to use LaTeX. Another option for LaTeX is to use Overleaf, which provides a shareable document interface that is housed online; this interface is convenient for collaborative writing and editing.

Once you have access to LaTeX, the easiest way to get started is to download the following packages at the following websites.
  • revtex package
  • natbib package

Citations

All documents will use BibTeX to create their reference. One master BIB-file will be continuously updated so that we do not need to redo work. I will update the file periodically.
  • master
  • walker_group
We will default to use the natbib package that is used for bibliographic management.

As an example, by performing a search in Google Scholar for the following title:

"Walker, T.W., A.N. Logia, G.G. Fuller. Multiphase flow of miscible liquids: jets and drops. Exp. Fluids. 56, 106 (2015)."

you will be given a number of links under the file name. For a particular link, if you choose "cite," a pop-up menu will appear. If you choose "BibTex" at the bottom, a new page will load with the following information.

@article{walker2015multiphase, title={Multiphase flow of miscible liquids: jets and drops}, author={Walker, Travis W and Logia, Alison N and Fuller, Gerald G}, journal={Experiments in Fluids}, volume={56}, number={5}, pages={1--14}, year={2015}, publisher={Springer} }

This information can then be copied and pasted into the BIB-file. Note that the article name is the same as our article naming convention.

Reports

All reports will be completed in LaTeX. We will work together to make a default template that will be used by everyone.

Presentations

All presentations can be completed in any format that is desired; however, format of materials given in the presentations will need to meet expectations. Note that LaTeX does have a presentation package called Beamer that is quite nice.

Posters

All posters need to be submitted to me for penultimate review at least 48-hours prior to the print-submission deadline. If you do not understand the word "penultimate," look it up.
